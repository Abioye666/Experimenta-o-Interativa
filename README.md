# Experimentação Interativa - Simulação de Rasgo

Uma simulação interativa de rasgo de malha wireframe usando Canvas2D e física Verlet.

## Características

- Simulação de física realista com integração Verlet
- Efeitos de rasgo progressivos
- Simulação de vento com turbulência
- Interações interativas (arrastar, cortar, rajadas)
- Controles ajustáveis em tempo real

## Como Usar

- **Click + Drag**: Arraste e rasgue a malha
- **Double Click**: Corte vertical
- **R**: Rajada de vento
- **C**: Corte circular
- **D**: Abrir/fechar painel de controles

## Deploy

Este projeto está hospedado no Vercel e pode ser acessado diretamente no navegador.

## Tecnologias

- HTML5 Canvas
- JavaScript (Vanilla)
- Física Verlet
- Simplex Noise
